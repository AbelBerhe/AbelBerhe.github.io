Projects

Here are some projects I completed as part of my studies, along with a brief description. To view the code for this projects, please check out my resume where you will find a link to my GitHub profile.

Country App

This desktop application, created using Python, is capable of reading data from a file and presenting a list of all countries worldwide. Additionally, it calculates the population density and the percentage of the world's total population for each country.

Mobile Quiz App

Built using Java, this mobile application functions as a quiz builder, and is capable of reading a file, loading data, and presenting a unique set of questions each time it runs.

Mobile Pizza App

As part of a class assignment, I have designed this mobile app focused on ordering pizzas with different sizes and toppings. This app is driven by data and saves orders to a database, which it then displays to users via a designated email. The app was created using Java.

SAR

As part of my work term, I have contributed to an ongoing desktop application project in my college that aims to monitor the progress of students. The primary objective of this application is to enable instructors to provide assistance to students. My primary contribution to this project was the design of the database.

Social Media Web App

This Social media Web app is designed to be responsive, fully functional, and driven by data. Users can create posts that are automatically timestamped, and the app features authentication, enabling users to log in with different roles for specific purposes. All data is sourced from and stored in a database, and the app is built using Laravel and PHP.

E-Portfolio Web App

This web application is completely functional and responsive, and is intended to exhibit my expertise and portfolio. It is built using CSS, HTML, and JavaScript.
